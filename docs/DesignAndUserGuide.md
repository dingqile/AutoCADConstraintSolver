## 设计思路

### 系统概述

AutoCADConstraintSolver 是一个参数化 CAD 约束求解器插件，基于 [NoteCAD](https://github.com/NoteCAD/NoteCAD) 项目移植开发。它采用代数方法解决几何约束问题，使设计师能够通过添加几何约束来精确控制图形的形状和尺寸。

#### 核心特性

- **参数化建模**：使用变量（参数）表示几何元素的位置和尺寸
- **约束驱动**：通过几何约束定义元素间的关系
- **实时求解**：参数修改后自动重新求解，图形即时更新
- **Newton-Raphson 迭代**：高效的数值求解方法

### 核心算法

#### 1. 约束系统数学模型

约束求解器的核心是将几何约束转化为代数方程组。假设我们有：

- **参数向量** `P = (p1, p2, ..., pn)` - 表示所有可变量
- **约束方程** `F(P) = 0` - 表示所有约束条件

求解目标：找到参数值 `P` 使得所有约束方程 `F(P) = 0` 成立。

#### 2. Newton-Raphson 迭代法

对于非线性方程组，使用 Newton-Raphson 迭代：

```
F(P) + J(P) * deltaP = 0
P_new = P_old + deltaP
```

其中：
- `F(P)` 是残差向量（所有约束方程的值）
- `J(P)` 是雅可比矩阵（约束方程对参数的偏导数）
- `deltaP` 是参数更新量

#### 3. 迭代流程

```
1. 初始化参数值
2. 计算残差 F(P) 和雅可比矩阵 J(P)
3. 求解线性方程组 J(P) * deltaP = -F(P)
4. 更新参数 P = P + deltaP
5. 检查收敛条件 ||F(P)|| < epsilon
6. 如果未收敛，返回步骤 2
```

#### 4. 约束方程示例

**点重合约束**：两点 (x1, y1) 和 (x2, y2) 重合
```
F1 = x1 - x2 = 0
F2 = y1 - y2 = 0
```

**水平约束**：直线两端点 y 坐标相等
```
F = y1 - y2 = 0
```

**距离约束**：两点间距离为 d
```
F = sqrt((x2-x1)^2 + (y2-y1)^2) - d = 0
```

### 约束求解策略

#### 1. 自由度分析 (DOF)

每个实体有不同的自由度：
- **点**：2 DOF (x, y)
- **直线**：4 DOF (两个端点)
- **圆**：3 DOF (圆心 x, y + 半径)
- **圆弧**：5 DOF (圆心 x, y + 半径 + 起始角 + 终止角)

约束消耗自由度：
- 重合约束：2 DOF
- 水平/垂直约束：1 DOF
- 距离约束：1 DOF
- 固定约束：2 DOF

#### 2. 方程排序与消元

为提高求解效率，方程系统会：
1. 识别可替换形式的方程（如 x = 5）
2. 替换参数以减少方程数量
3. 按依赖关系排序方程

#### 3. 收敛判定

```
收敛条件：max(|Fi(P)|) < epsilon
发散检测：如果 ||deltaP|| 过大，跳过该步
最大迭代：防止无限循环
```

## 用户使用说明

### 安装配置

#### 系统要求

- **.NET Framework 4.8** 或 **.NET 8.0**
- **Visual Studio 2022** 或更高版本
- **AutoCAD 2021-2026**

#### 编译步骤

```bash
# 1. 克隆仓库
git clone https://github.com/dingqile/AutoCADConstraintSolver.git
cd AutoCADConstraintSolver

# 2. 使用 Visual Studio 打开 AutoCADConstraintSolver.sln

# 3. 选择目标 AutoCAD 版本配置（如 Debug|2024）

# 4. 编译生成 AutoCADConstraintSolver.dll
```

#### 部署到 AutoCAD

```bash
# 方法 1：NETLOAD 命令
# 在 AutoCAD 中输入：NETLOAD
# 选择编译生成的 DLL 文件

# 方法 2：添加到启动目录
# 将 DLL 复制到 AutoCAD 支持文件搜索路径
```

### 基本操作

#### 1. 启动插件

```
命令: ConstraintSolver 或 CS
快捷键: 自定义
```

启动后，插件会加载约束工具栏和面板。

#### 2. 创建实体

| 命令 | 快捷键 | 功能 |
|------|--------|------|
| CSLine | L | 绘制直线 |
| CSCircle | C | 绘制圆 |
| CSArc | A | 绘制圆弧 |
| CSEllipse | E | 绘制椭圆 |

#### 3. 选择实体

```
命令: CSSelect
快捷键: S
```

- **单击**：选择单个实体
- **框选**：框选多个实体
- **按住 Shift**：添加到选择集
- **CSAll**：全选

#### 4. 删除实体/约束

```
命令: CSDelete
快捷键: Delete
```

### 约束应用指南

#### 几何约束

##### 重合约束 (Coincident)
将两个点约束到同一位置。

```
步骤：
1. 运行 CSCoincident 命令
2. 选择第一个点
3. 选择第二个点
```

**应用场景**：
- 连接线条端点
- 确定圆心位置

##### 水平/垂直约束
使直线水平或垂直。

```
步骤：
1. 运行 CSHorizontal 或 CSVertical 命令
2. 选择要约束的直线
```

##### 平行/垂直约束
两直线平行或垂直。

```
步骤：
1. 运行 CSParallel 或 CSPerpendicular 命令
2. 选择第一条直线
3. 选择第二条直线
```

##### 相切约束
使直线与圆/弧相切。

```
步骤：
1. 运行 CSTangent 命令
2. 选择圆或弧
3. 选择直线
```

#### 尺寸约束

##### 距离约束
约束两点之间的距离。

```
步骤：
1. 运行 CSDistance 命令
2. 选择第一个点
3. 选择第二个点
4. 输入距离值
```

**注意**：AutoCAD 测量的是实际距离，选择时拖动方向决定约束方向。

##### 长度约束
约束直线段的长度。

```
步骤：
1. 运行 CSLength 命令
2. 选择直线
3. 输入长度值
```

##### 直径约束
约束圆的直径。

```
步骤：
1. 运行 CSDiameter 命令
2. 选择圆
3. 输入直径值
```

##### 角度约束
约束两直线间的角度。

```
步骤：
1. 运行 CSAngle 命令
2. 选择第一条直线
3. 选择第二条直线
4. 输入角度值（度）
```

## 示例教程

### 绘制矩形

#### 目标
使用约束方法绘制一个 100x60 的矩形。

#### 步骤

1. **启动插件**
   ```
   命令: CS
   ```

2. **绘制四条边**
   ```
   命令: CSLine
   - 第一条: (0,0) -> (100,0)
   - 第二条: (100,0) -> (100,60)
   - 第三条: (100,60) -> (0,60)
   - 第四条: (0,60) -> (0,0)
   ```

3. **添加水平/垂直约束**
   ```
   命令: CSHorizontal
   - 选择第一条线 (底部)
   - 选择第三条线 (顶部)
   
   命令: CSVertical
   - 选择第二条线 (右边)
   - 选择第四条线 (左边)
   ```

4. **添加长度约束**
   ```
   命令: CSLength
   - 选择第一条线，输入: 100
   
   命令: CSLength
   - 选择第二条线，输入: 60
   ```

5. **添加重合约束（连接端点）**
   ```
   命令: CSCoincident
   - 选择 L1 终点和 L2 起点
   - 选择 L2 终点和 L3 起点
   - 选择 L3 终点和 L4 起点
   - 选择 L4 终点和 L1 起点
   ```

6. **固定一个角点**
   ```
   命令: CSFixation
   - 选择左下角点 (0,0)
   ```

7. **验证**
   - 拖动任意端点，矩形保持直角和指定尺寸

### 绘制相切圆

#### 目标
绘制一个与两条相交直线相切的圆。

#### 步骤

1. **绘制相交直线**
   ```
   命令: CSLine
   - L1: (0,0) -> (100,50)
   - L2: (0,100) -> (100,50)
   ```

2. **绘制圆**
   ```
   命令: CSCircle
   - 圆心: 近似在交点上方，如 (50,80)
   - 半径: 约 30
   ```

3. **添加相切约束**
   ```
   命令: CSTangent
   - 选择圆
   - 选择 L1
   
   命令: CSTangent
   - 选择圆
   - 选择 L2
   ```

4. **调整圆位置**
   - 现在圆会自动调整到与两直线相切的位置

### 绘制对称图形

#### 目标
绘制关于某轴对称的两个圆。

#### 步骤

1. **绘制对称轴**
   ```
   命令: CSLine
   - 轴线: (50,0) -> (50,100)
   ```

2. **绘制第一个圆**
   ```
   命令: CSCircle
   - 圆心: (25,50)
   - 半径: 15
   ```

3. **绘制第二个圆**
   ```
   命令: CSCircle
   - 圆心: (75,50)
   - 半径: 15
   ```

4. **添加约束**
   ```
   命令: CSConcentric
   - 选择两圆（使半径参数关联）
   
   命令: CSCoincident
   - 选择圆1 圆心关于轴线的对称点
   - 选择圆2 圆心
   ```

5. **固定轴线**
   ```
   命令: CSVertical
   - 选择轴线
   ```

6. **验证对称性**
   - 修改任一圆的半径，另一圆自动更新

## 附录

### A. 约束自由度消耗表

| 约束类型 | 消耗 DOF | 说明 |
|----------|----------|------|
| 点重合 | 2 | 两点 x,y 坐标相等 |
| 水平 | 1 | y1 = y2 |
| 垂直 | 1 | x1 = x2 |
| 平行 | 1 | 方向向量叉积 = 0 |
| 垂直 | 1 | 方向向量点积 = 0 |
| 相切 | 1 | 距离 = 半径 |
| 中点 | 2 | x = (x1+x2)/2, y = (y1+y2)/2 |
| 同心 | 2 | 圆心坐标相等 |
| 距离 | 1 | sqrt((x2-x1)^2+(y2-y1)^2) = d |
| 长度 | 1 | sqrt((x2-x1)^2+(y2-y1)^2) = L |
| 直径 | 1 | 2r = D |
| 角度 | 1 | atan2(cross, dot) = theta |
| 固定 | 2 | x = X, y = Y |
| 等长 | 1 | |L1| = |L2| |
| 等半径 | 1 | r1 = r2 |

### B. 求解状态解读

| 状态 | 含义 | 处理方法 |
|------|------|----------|
| OKAY | 求解成功 | 正常 |
| DIDNT_CONVERGE | 未收敛 | 检查约束是否冲突或不足 |
| REDUNDANT | 冗余约束 | 移除多余约束 |
| POSTPONE | 延迟求解 | 等待依赖项 |
| JUMP | 参数跳跃过大 | 调整初始值 |
| INTERNAL_FAILURE | 内部错误 | 简化问题 |

### C. 常见问题

**Q: 拖动时图形不更新**
A: 检查是否启用了自动求解，运行 CSSolve 手动触发。

**Q: 约束显示为灰色**
A: 约束被判定为冗余或过约束，检查是否有冲突约束。

**Q: 求解失败 (DIDNT_CONVERGE)**
A: 
1. 检查自由度是否不足（欠约束）
2. 检查约束是否冲突（过约束）
3. 调整实体初始位置
